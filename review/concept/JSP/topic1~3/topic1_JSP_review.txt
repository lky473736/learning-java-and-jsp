- 인터넷이란?
	- 전 세계에 걸쳐 컴퓨터끼리 원거리 접속이 가능한 TCP/IP를 이용하는 네트워크
	- "인터넷에서 제공하는 서비스" 중 가장 많이 사용하는 서비스 : 웹
		- 정보 교환 서비스 (정보 교환을 목적)
		- "하이퍼텍스트" 형식의 문서를 이용하여 정보를 교환하는 서비스 (HTML) -> 하이퍼링크를 통하여 다른 문서로 이동 가능 
	- 인터넷에서 제공하는 서비스
		- 이메일
		- 웹
		- 텔넷
		- 파일 송수신
		- ...
- 웹브라우저
	- 웹 서비스를 이용하는 프로그램
	- HTML, CSS, Javascript를 해석
- URL (Uniform Resource Locator)
	- 인터넷에서 자원의 위치를 표현하는 형식
		- 웹
		- 영상
		- 미디어
	- "서비스프로토콜://주소:포트/자원명" 형식으로 구성
	- 웹의 주소를 지칭하는 것이 아니다. 무조건 http를 적는 것이 아님
		- ftp://~~~~ : ftp서비스로 파일 주고받을 때
		- telnet://~~~~ : 원격 접속 서비스

-------

인터넷은 가장 큰 네트워크이며, 데이터 통신에서 배운 토대로 말하자면 라우터와 라우터의 상호 연결이라고 볼 수 있다.
TCP/IP를 이용하며, TCP는 여기서 Transmission, Control, Protocol의 줄임말이다. IP는 넘어가자.
인터넷에서 제공하는 여러 서비스가 있는데, 메일 서비스나 파일 저장 서비스... 다양하지만 가장 많이 사용하는 서비스는 당연히 웹이다.
일종의 정보 교환 서비스이며, 하이퍼텍스트 형식의 문서를 이용하여 정보를 교환한다. (HTML을 의미)

웹브라우저는 웹 서비스를 이용하게 만들어주는 프로그램으로써, HTML, CSS, Javascript를 해석하는 해석기의 역할을 한다.

URL은 웹의 주소를 지칭하는 것이 아니다. http와 같은 웹 주소 뿐만이 아닌 ftp, telnet 등 다양하다. URL은 여기서 인터넷의 자원의 위치를 표현하는 형식을 의미한다.
형식은 서비스프로토콜://주소:포트/자원명 식이다.
