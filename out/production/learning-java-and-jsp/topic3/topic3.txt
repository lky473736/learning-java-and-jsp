JAVA(황희정) topic 3

- JDK 설치
    - 자바 개발 환경을 구축하기 위해 JDK를 구축해야 함
    - JDK
        - 자바를 개발하기 위한 컴파일러, 디버거, 기본 라이브러리 포함
        - 오라클이 자바 라이센스 변경 -> 무료로 사용하는 OpenJDK 사용하는 것이 좋음
        - OpenJDK랑 오라클 JDK랑 똑같음, 하지만 OpenJDK는 설치가 불편해서 Eclipse Temurin을 많이 사용함

- 통합 개발 환경 (IDE / Integrated Development Environment)
    - 이클립스
    - IntelliJ IDEA
        - 동일한 개발 플랫폼에 다른 언어에 특화된 여러 제품 존재 (Pycharm 등)
        - gradle, github, docker, 데이터베이스 등 개발자한테 필요한 도구 사용하기 편하게 통합됨
